# JavaScript Portfolio(FAQ & Modal UI)

## 概要
JavaScriptのDOM操作とイベント制御の理解を深めるために制作しました。
実務を想定し、ユーザーが直感的に操作できるUI設計を意識しています。
各機能は再利用性と可読性を考慮して関数化しています。


## 使用技術
- HTML
- CSS
- JavaScript

## 実装機能
### ハンバーガーメニュー
- クラスの付け外しで開閉制御
- 背景クリックで閉じる
- スクロール制御（no-scroll）
- 関数化して可読性を向上

### FAQアコーディオン
- 1つだけ開く仕様
- 再クリックで閉じる実装
- boolean(true/false)を使った制御

### モーダルウィンドウ
- ボタンクリックで表示
- 背景クリックで閉じる
- Contactリンクからの連携
- ナビを閉じてからモーダルを開く制御

## 工夫した点
- 開閉処理を共通関数にまとめ、保守性を向上
- 三項演算子とif文を使い分け、コードの可読性を意識
- ナビゲーションとモーダルの動作が干渉しないよう制御順を設計
- UIの自然な挙動（背景クリックで閉じるなど）を意識

## 学んだこと
- classListの操作
- event.preventDefault()の役割
- booleanを使ったロジック制御
- コードの整理（リファクタリング）
