# JavaScript Portfolio(FAQ & Modal UI)

## 概要
JavaScriptのDOM操作とイベント制御の理解を深めるために制作しました。

実務を想定し、ユーザーが直感的に操作できるUI設計を意識しています。
各機能は再利用性と可読性を考慮して関数化しています。

## 使用技術
- HTML
- CSS
- JavaScript

## 実装機能
### ハンバーガーメニュー
- クラスの付け外しで開閉制御
- 背景クリックで閉じる仕様
- スクロール制御（no-scroll）
- 処理の関数化による保守性向上

### FAQアコーディオン
- 1つのみ開く仕様
- 再クリックで閉じる実装
- boolean(true/false)による状態管理

### モーダルウィンドウ
- ボタンクリックで表示
- 背景クリックで閉じる
- Contactリンクからの連携
- ナビを閉じてからモーダルを開く制御

## 工夫した点
- 重複処理を関数化し、コードを整理
- UI同士が干渉しないようイベント処理順を設計
- モバイル表示を意識したレスポンシブ対応
- viewport設定による適切なスマホ表示制御

## 対応可能なこと
- JavaScriptによるUI挙動の実装（ハンバーガー・モーダル・アコーディオン）
- classListを用いた状態管理
- booleanによるロジック制御
- event.preventDefault() を用いたリンク制御
- 関数化によるコードの整理・可読性向上
- レスポンシブ対応（viewport設定含む）

- CSSアニメーションの最適化
- レスポンシブ対応の強化
